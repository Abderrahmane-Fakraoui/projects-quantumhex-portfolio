# QuantumHex Portfolio

A personal portfolio / landing page for **Abderrahmane Fakraoui — QuantumHex**, a Moroccan competitive programmer training for IOI 2027/2028.

## Features

- Balanced visual theme: light base, strong ink, Codeforces-inspired blue/green accents
- Live public Codeforces stats for `QuantumHex`
- Personal story, goals, skills, links, and project highlight
- Animated algorithm field background
- Animated profile image panel
- GitHub Pages workflow included

## Local development

Run these commands on your own computer:

```bash
npm i
npm run dev
```

## Deploy on GitHub Pages

1. Create a repository on GitHub.
2. Upload all project files.
3. Go to **Settings → Pages**.
4. Choose **GitHub Actions** as the source.
5. Push to `main` and wait for the deploy action.

## Customize

Open `src/main.tsx` and update:

- Email when ready
- Discord link if you want a direct Discord user link
- Story text and project descriptions

## Note about Codeforces API key

The public Codeforces endpoints used here do not require a signed API key. The key is included in the source as requested, but the app fetches public stats directly.
