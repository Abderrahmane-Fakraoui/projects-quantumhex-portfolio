# QuantumHex Portfolio

Personal portfolio for **Abderrahmane Fakraoui** ([@QuantumHex](https://codeforces.com/profile/QuantumHex)) — a kinetic graph-field landing page for a Moroccan competitive programmer training toward IOI 2027/2028.

## Features

- Interactive cursor-driven graph field inspired by algorithm visualization
- Animated constellation hero, glowing skill orbit, sticky project cards
- Responsive layout for desktop, tablet, and mobile
- Accessible focus states and reduced-motion support
- Static Vite build, ready for GitHub Pages

## Run locally

```bash
npm i
npm run dev
```

## Build

```bash
npm run build
npm run preview
```

## Deploy to GitHub Pages

1. Create a GitHub repository.
2. Push all files in this project.
3. Go to **Settings -> Pages**.
4. Under **Build and deployment**, choose **GitHub Actions**.
5. Push to the `main` branch. The workflow deploys automatically.

## Customize

- Update links in `src/main.tsx`:
  - Codeforces profile
  - GitHub profile
  - Email address
- Update project cards in the `projects` array.
- Update skill tags in the `skills` array.
